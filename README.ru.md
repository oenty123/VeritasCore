# VeritasCore — честный гейт безопасности для Python

**VeritasCore** — это контрактно-ориентированный статический анализатор безопасности для Python, который не требует внешних зависимостей (только стандартная библиотека Python 3.10+) и даёт **почти нулевой уровень ложных срабатываний**.

Вместо того чтобы навязывать жёсткие правила, VeritasCore **изучает контракты защиты из вашего кода** (например, что вы оборачиваете `os.system` в `shlex.quote`) и требует соблюдения этих контрактов в новом коде. Там, где доказательств недостаточно, инструмент честно отвечает `unknown` и **не блокирует** коммит.

---

## Основные возможности

- **Четыре состояния**: `guarded` (защищён), `unguarded` (уязвим), `safe` (константа), `unknown` (недостаточно данных) — минимизация шума.
- **Принцип Zero**: никогда не утверждаем больше, чем можем доказать.
- **Автофикс**: интеллектуальная вставка guard-ов с учётом контекста и импортов.
- **Межфайловый анализ** (опционально) — отслеживание taint через импорты.
- **Кросс-проектное обучение** — накопление контрактов из разных проектов в базе знаний.
- **Веб-дашборд** — локальный интерфейс для визуального аудита.
- **Сканер секретов** — высокоточное обнаружение ключей, токенов, паролей.
- **Производительность** — сканирование Home Assistant (3M строк) за 15 секунд.
